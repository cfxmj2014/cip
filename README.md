# cip-cli
this is a continuous integration platform cli

### Installation

require: [Node.js](https://nodejs.org/en/) (>=4.x), npm version 3+ and [Git](https://git-scm.com/).

``` bash
$ npm install -g cip-cli
```

### Usage

``` bash
$ cip init <project-name>
```

Example:

``` bash
$ cip init my-project
```
